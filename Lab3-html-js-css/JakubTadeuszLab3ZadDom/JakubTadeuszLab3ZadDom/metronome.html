How to use:

To use the metronome, you need to set the choosen BPM (Beats per minute) using the input field or tempo tapping button
Then, you need to specify the beats amount in one bar using the and buttons.
By clicking the beat indicators, you can change sound between the drum beat and cymbal. Both of the beat sounds comes from the Roland TR-808 drum machine.
- Default drum beat sound
- Cymbal sound
After setting up the metronome, click button to begin.
The red color highlights the current beat. While the metronome is working, you can freely change the beats amount or the beats sound.
To change BPM, you need to restart using the button.
