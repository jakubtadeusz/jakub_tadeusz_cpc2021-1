How to use:
Click on the choosen string and fret to set the marker.
Click above the fretboard to set the string marker to open. If the string is set to open (O), next click will change string state to muted (X).
If you want to save the chord, type your chord name into the textbox and clickbutton.
After saving, chord will be visible in the Guitar chords page.
If you want to change the chord, use save function with the name of the chord you would like to change, this will override the original chord.
To save the chord as an image, just simply save the image using right mouse button
